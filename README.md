# Système de gestion de tickets

Ce projet a été réalisé personnelelement inspiré par mon apprentissage en entreprise.  
Il simule une interface de support permettant de créer, afficher et filtrer des tickets incidents.

## 🧰 Technologies utilisées

- HTML / CSS
- PHP
- MySQL

## ⚙️ Fonctionnalités

- Ajout de tickets via formulaire
- Affichage de tous les tickets
- Filtrage des tickets par statut : `Ouvert`, `En cours`, `Résolu`

## 🎯 Objectif pédagogique

- Gérer des données dynamiquement en PHP
- Manipuler des formulaires et interactions backend
- Comprendre le fonctionnement d’un système de support simple

## 📁 Organisation des fichiers

- `/index.php` → Affichage et filtrage des tickets  
- `/ajout_ticket.php` → Formulaire pour créer un ticket  
- `/includes/` → fichiers annexes (connexion DB, fonctions, etc.)
